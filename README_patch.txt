Este app.js deja Supabase como fuente única de datos.

- No contiene valores fijos de gastos.
- Resumen, Control de Proyecto, Proveedores, Caja e IVA, Balance y Reportes se calculan desde public.gastos_junquillar_app.
- Si eliminas todos los gastos en Supabase, todas las páginas quedan en cero o sin registros.
- Mantiene PROJECT_BUDGET solo como presupuesto referencial para calcular porcentaje de avance.
